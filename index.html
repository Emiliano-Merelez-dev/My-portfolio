Mis Proyectos

e-commerce App

Este es un proyecto de e-commerce que permite a los usuarios navegar, seleccionar y comprar productos de manera sencilla y eficiente. El sistema proporciona una experiencia de usuario fluida con funcionalidades como el registro y inicio de sesión mediante JWT, gestión de carrito de compras, proceso de pago y visualización de productos filtrados por categorías. Además, incluye un panel de administración para gestionar productos, categorías, y pedidos. Herramientas usadas: Next.js, tailwind css y postgresSQL

Calendar App

Este proyecto es un calendario interactivo diseñado para facilitar la gestión de eventos y tareas. Permite a los usuarios agregar, editar y eliminar eventos de manera sencilla. cuenta con un registro y inicio de sesion por el cual pase por un proceso de authenticacion. Herramientas usadas: React, css, html, JavaScript, node y MongoDB.

Pizzería App

Este es un proyecto de pizzería en línea simple, diseñado para ofrecer a los usuarios una experiencia rápida y fácil al explorar el menú de pizzas y agregar sus opciones favoritas. La aplicación cuenta con tres secciones principales: la página de inicio, el menú de pizzas y una lista de favoritos, donde los usuarios pueden ver y seleccionar sus pizzas preferidas. Herramientas usadas: react, css y html.

GifExpert App

Es una aplicación web que permite a los usuarios buscar y visualizar gifs animados de manera rápida y sencilla. Los usuarios pueden ingresar términos de búsqueda, y la aplicación recupera y muestra los gifs más relevantes relacionados con la consulta. real. Herramientas usadas: react, fetch hacia una API, css y html.
